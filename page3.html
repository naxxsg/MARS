INTERFACES Y LENGUAJES
DE PROGRAMACIÓN PARA DISEÑO WEB

Principios de html, desde definicion del sitio web hasta las primeras bases de html, css y java script.

¿Qué es un sitio web?

Un sitio web es un espacio digital en internet, constituido por diversos documentos que contienen texto, fotos, videos, etc. que se estructuran para dar presencia en línea a una marca, empresa, persona, negocio, producto o servicio. Un sitio web es un conjunto de páginas web bien organizadas. Una página web es un documento electrónico que puede estar creado con código HTML y es accesible desde cualquier navegador.

Un sitio web necesita un alojamiento llamado hosting y se relaciona a un dominio, que es el nombre del sitio, que le permitirá ser visibles en internet a tráves de cualquier punto de acceso.

Dominio

Es la dirección y nombre de tu sitio web.

Se recomienda escoger un nombre sencillo de escribir y recordar junto con una extensión formal (.com, .net, .org)

Hosting

Es el espacio físico dónde se almacena tu sitio web, puede ser una computadora o un servidor que se contrata en una empresa de alojamiento con distintos precios.

HTML

HyperText Markup Naguage

Traduce a Lenguaje de Marcas de HyperTexto. HTML5, es la revisión 5 del estándar creado en 1991 por Tim Berners-Lee y contenia originalmente 22 elementos.

HTML es un markup language, lo que significa que está escrito con códigos que puede leer una persona sin necesidad de compilarlo previamente. El texto del HTML está "marcado" con estos códigos para dar instrucciones al navegador web sobre cómo mostrar el texto.

CSS

Cascading Style Sheets

Es el código que da estilo al contenido web. Al igual que el HTML, CSS no es un lenguaje de programación. Tampoco es un lenguaje de marcas. CSS es un lenguaje de hojas de estilo, permitiendo diseñar los elementos HTML de forma selectiva.

El selector es el elemento HTML al comienzo del conjunto de reglas, define los elementos a los que se les aplicar[a estilo, seguido por una declaración y finalmente con propiedades.

JavaScript

Es un lenguaje de secuencias de comandos o de programación que le permite implementar funciones complejas en páginas web. Permite mostrar actualizaciones de contenido oportunas tales como mapas interactivos, animaciones 2D, gráficos en 3D, jukeboxes de video con desplazamiento, etc.








¿Qué es un editor?


Un editor de texto es un programa informático que permite crear y modificar archivos digitales compuestos únicamente por textos sin formato, el programa lee el archivo e interpreta los bytes leídos según el código de caracteres que usa el editor.

Los editores son incluidos en el sistema operativo o en algún paquete de software instalado, y se usan uando se deben crear o modificar archivos de texto como archivos de configuración, lenguaje de programación interpretado o el código fuente de algún programa.
