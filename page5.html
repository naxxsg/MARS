Estructura de una
Página Web

Introducción a la estructura html de una página web + mini guia de etiquetas básicas para principiantes.

Documentación Web

La documentación de un sitio web previene rrores y asegura el correcto uso de la plataforma en el futuro.

Estructura de Archivos

Describe la organización de las carpetas que contienen los archivos del desarrollo front end y back end. Usualmente se clasifican según su extensión, por ejemplo .html, .css o .js.

Atributos Personalizados

Para programar funcionalidades especiales es posible usar atributos personalizados en HTML5, los cuales son reconocibles gracias al prefijo "data-". Esta sección incluye una descripción de estos atributos. Además, se entrega un detalle de las partes en que se ha implementado JavaScript, sus funciones en el sitio y cómo este se conecta al HTML.

imágenes

Para añadir imágenes a una página web hay que escribir el atributo "src", el cual indica la ruta de la imagen. Si el archivo de imagen se encuentra en la misma carpeta que la web sera más sencilla su inscripción.

src="/carpeta/archiv.jpg"

Tambien puedes usar la dirección completa para insertar una imagen.

src="https://web.com/carpeta/archivo.jpg"

Para indicar el ancho y alto de imagen se usan las etiquetas height y width.

img src="archivo.jpg" width="300" height="150"

Enlaces

Para indicar enlaces ya sea a otra página del sitio web o a un enlace externo, se usa la etiqueta href.

a href="https://web.com/"

Listas

Entre listas ordenadas según una orden en especifico, o listas no ordenadas, hay una etiqueta asignada para cada tipo de listas, la etiqueta "ol" es para listas ordenadas, y la etiqueta "ul" para las no ordenadas. Cada elemento de la lista es creado con la etiqueta "li"

Formatos de Texto

La etiqueta "b" sirve para resaltar en negrita las palabras seleccionadas entre ella. La etiqueta "i" para italicas y la etiqueta "u" para subrayados.

A su vez hay etiquetas que cuentan con el mismo efecto que las presentadas previamente, la etiqueta "strong" equivale a "b", la etiqueta "em" a "i" y la etiqueta "ins" a "u".
